To run the program, complies the files and run the ChatGUIServer.java file, followed by any number of runs of ChatGUIClient.java

ChatGUIServer:
When first run, a window should appear asking for a port number to host the server on, if left blank it will default to port 14002

ChatGUIClient:
When first run, a window shoudl appear asking for a port number and IP address of the server
If the port number is left blank, it will default to 14002
If the IP address is left blank, it will default to the localhost ip
